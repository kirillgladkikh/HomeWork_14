# Проект "Домашнее задание 14"

## Описание:

Проект "Домашнее задание 15.1" - это код на Python покрытый тестами.
Работа над магазином: в этой домашней работе оптимизируем работу с данными путем добавления магических методов.

Реализованы:
 - классы Product и Category. 
 - добавлены геттеры и сеттеры
 - добавлены Магические методы

### ДЗ 15.1

Строковое представление объектов:
- Для класса Product переопределен метод __str__, который возвращает строку: Название продукта, X руб. Остаток: X шт.
- Для класса Category переопределен метод __str__, который возвращает строку: Название категории, количество продуктов: X шт.
- Метод __str__ рассчитывает общее количество товаров на складе (quantity) для каждого продукта в приватном атрибуте products.

Магический метод сложения:
- Для класса Product переопределен магический метод сложения __add__, который принимает два аргумента: self и второй объект.
- Метод возвращает сумму произведений цены на количество у двух объектов.

### ДЗ 14.2

Режимы доступа к атрибутам:
- Атрибут списка товаров класса Category имеет приватный режим доступа.
- Реализован метод add_product() для добавления продукта в атрибут products.
- Метод add_product() принимает на вход аргументы self и продукт для добавления.
- В приватный атрибут products продукт добавляется через метод append.
- Метод add_product() не возвращает значения.
- Метод add_product() в классе Category прибавляет 1 к класс-атрибуту «счетчик продуктов».

Геттер:
- Реализован геттер для приватного атрибута products с использованием декоратора @property.
- Геттер возвращает строку со всеми продуктами в приватном атрибуте products.
- Имя метода-геттера — products.
- Каждый продукт в геттере сформирован как строка строго по шаблону: "Название продукта, X руб. Остаток: X шт.\n".

Класс-метод:
- Реализован класс-метод new_product() с помощью декоратора @classmethod.
- Метод new_product() принимает на вход два аргумента — cls и продукт для создания в виде словаря — и возвращает экземпляр класса на основе данных словаря.

Работа с атрибутом цены:
- Атрибут «цена» класса Product имеет приватный режим доступа.
- Реализован геттер для атрибута «цена» с использованием декоратора @property. Геттер возвращает значение приватного атрибута цены.
 -Название метода-геттера — price.
- Для приватного атрибута «цена» реализован сеттер в классе Product с использованием декоратора @price.setter.
- Сеттер принимает два аргумента — self и новое значение цены.
- В сеттере реализована проверка на положительное значение новой цены.
- Если новое значение цены положительное, то в атрибуте обновляется значение цены.
- Если новое значение цены не положительное, то новое значение приватному атрибуту цены не присваивается и выводится сообщение «Цена не должна быть нулевая или отрицательная» в консоль.
- Сеттер не возвращает никаких значений.
- Название метода-сеттера — price.

## Содержит функции:

### модуль [categories_products.py](src%2Fcategories_products.py):

class Product - Класс для представления товара.
    :param name: название товара
    :param description: описание товара
    :param price: цена товара (может содержать дробную часть)
    :param quantity: количество товара в наличии

class Category - Класс для представления категории товаров.
    :param name: название категории
    :param description: описание категории
    :param products: список товаров в категории (по умолчанию пустой)
